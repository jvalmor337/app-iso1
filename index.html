AD-Check ISO

Assistent educatiu compacte per a preparar una pràctica de Windows Server i Active Directory en 1r ASIR.

RA: Windows Server · AD DS · DNS · Permisos

1. Configuració del servidor

Comprova que el controlador de domini està ben identificat i que la configuració IP/DNS és coherent.

Un controlador de domini hauria de tindre IP fixa.

Normalment ha d'apuntar a ell mateix o al DNS intern del domini.

Per a pràctiques d'aula és habitual usar un domini intern, com asir.local o aula.local.

2. Client Windows i connectivitat

Abans d'unir el client al domini, cal comprovar DNS, Firewall i ping bidireccional.

Per unir-se al domini, el client ha d'usar com a DNS el servidor AD.

No són regles de Firefox, sinó regles d'entrada del Firewall.

Cal comprovar connectivitat en els dos sentits: client → servidor i servidor → client.

Pista docent: si falla el ping, abans de tocar Active Directory cal revisar IP, màscara, adaptador de VirtualBox i Firewall de Windows.

3. Estructura d'Active Directory

Defineix una estructura mínima d'OU, usuaris i grups per administrar el domini de forma ordenada.

4. Carpeta compartida i permisos

Revisa si els permisos s'apliquen de forma adequada i segura.

En administració real és millor aplicar permisos a grups, no usuari per usuari.

5. Revisió automàtica

Prem el botó per rebre feedback abans de fer la pràctica en VirtualBox.

6. Resum per a la memòria de pràctica

Aquest resum es pot copiar i enganxar en la memòria de l'activitat.
