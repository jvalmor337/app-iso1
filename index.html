AD-Check ISO

Assistent educatiu per a revisar una pràctica de Windows Server i Active Directory en 1r ASIR.

AD DS · DNS · Ping · Firewall · Permisos · CSV local

1. Configuració del servidor

L'alumnat introdueix les dades tècniques de la seua pràctica. No apareix cap llista d'alumnes; cada alumne/a escriu el seu codi o inicials.

Aquest codi permet al professorat identificar el registre en el full de càlcul local.

Un controlador de domini hauria de tindre IP fixa.

Normalment ha d'apuntar a ell mateix o al DNS intern del domini.

Per a pràctiques d'aula és habitual usar un domini intern, com asir.local o aula.local.

2. Client Windows i connectivitat

Abans d'unir el client al domini, cal comprovar DNS, Firewall i ping bidireccional.

Per unir-se al domini, el client ha d'usar com a DNS el servidor AD.

No són regles de Firefox, sinó regles d'entrada del Firewall de Windows.

Cal comprovar connectivitat en els dos sentits: client → servidor i servidor → client.

Si falla el ping, abans de tocar Active Directory cal revisar IP, màscara, adaptador de VirtualBox, DNS i Firewall de Windows.

3. Estructura d'Active Directory

Defineix l'organització bàsica d'OU, usuaris i grups de seguretat.

4. Carpeta compartida i permisos

Revisa si els permisos s'apliquen de forma adequada i segura.

En administració real és millor aplicar permisos a grups, no usuari per usuari.

5. Revisió automàtica i exportació local

Analitza la configuració, guarda el registre en aquest navegador i descarrega el CSV per obrir-lo en Excel, LibreOffice Calc o Google Sheets.
